## 💻 About This Project (The Portfolio)

This repository contains the source code for my personal portfolio website. It was designed from scratch to mimic the intuitive and sleek UI of Spotify, utilizing a dark color palette, monospace typography, and interactive media components.

### ✨ Key Features

- **Glassmorphism Navigation:** A custom, frosted-glass fixed navigation bar that dynamically highlights the active section using the Intersection Observer API.
- **Full-Screen Snap Scrolling:** CSS-native scroll snapping for a premium, app-like browsing experience.
- **Interactive Project Carousel:** An expandable, draggable slider built with Owl Carousel to showcase development projects.
- **Fully Responsive:** Fluid design that scales perfectly across desktop, tablet, and mobile devices.

### 🛠 Tech Stack Built With
- **HTML5 & CSS3** (Custom animations, flexbox, grid, glassmorphism)
- **Vanilla JavaScript** (DOM manipulation, scroll tracking)
- **jQuery & Owl Carousel 2** (Interactive sliders)
